## Development

First, rename `.env.example` to `.env.local`, and fill out `RF_API_KEY` with your Roboflow API key.

Run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
